## Technical References

### Atari ST Architecture
- **Processor**: Motorola 68000
- **Byte Order**: Big-endian
- **File System**: TOS (The Operating System)

### MIDI Format
- Standard MIDI File format may be referenced
- Notator uses pattern-based MIDI sequencing

## Analysis Tools

### Hex Editors
- **HxD** (Windows)
- **Hex Fiend** (macOS)
- **Bless** (Linux)
- **010 Editor** (Cross-platform, commercial)

### Binary Analysis
- **xxd** (Unix command-line)
- **hexdump** (Unix command-line)
- Custom Python/Node.js scripts
